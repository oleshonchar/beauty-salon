## Technologies used
* html5
* css3
* bootstrap (grid)

* SASS

## Build and run
Requirements:
* nodejs
* npm
* gulp
* git
How to run:

    git clone git@github.com:oleshonchar/beauty-salon.git
    cd beauty-salon
    npm install
    gulp serve
